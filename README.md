# NVC-AK Southern Boreal Alpine
Ordination and cluster analysis of vegetation plots above treeline in the Southern Boreal zone.

Script 01 requires a public connection to the AKVEG Database. You can also execute script 02 or see the results in the following [public Google Drive folder](https://drive.google.com/drive/folders/1mAcHTobnpnostxxnYV74gJgY1mKQxkq2?usp=sharing).
